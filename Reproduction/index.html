Reproduction Guide

GDKVM: Echocardiography Video Segmentation

Back to Main Page

1. Environment Setup

The code is implemented using PyTorch. We recommend using Anaconda to manage the environment. 

conda create -n gdkvm python=3.8
conda activate gdkvm
pip install torch==1.13.0 torchvision==0.14.0 --extra-index-url https://download.pytorch.org/whl/cu116
pip install -r requirements.txt

2. Data Preparation

We use CAMUS and EchoNet-Dynamic datasets. Please download them from their official websites and organize the data as follows: 

./data
├── CAMUS
│   ├── training
│   └── testing
└── EchoNet-Dynamic
    ├── FileList.csv
    ├── VolumeTracings.csv
    └── Videos

3. Training

To train the GDKVM model on the CAMUS dataset, run the following command. The checkpoints will be saved in the ./checkpoints directory. 

python train.py --dataset camus --batch_size 8 --epochs 100 --lr 1e-4 --save_dir ./checkpoints/camus_gdkvm

4. Inference & Evaluation

To evaluate the pre-trained model, run: 

python test.py --dataset camus --resume ./checkpoints/camus_gdkvm/best_model.pth --eval_metrics DSC HD95
