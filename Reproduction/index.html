1. 环境配置

1.1 关于 uv

uv 是一个极速的 Python 包管理器和项目管理工具，由 Rust 编写。它旨在替代 pip、pip-tools 和 virtualenv，提供比传统工具快 10-100 倍的速度。

1.2 安装 uv

如果服务器无法直接连接外网或没有 root 权限，推荐直接使用 pip 安装：

pip install uv

1.3 常见问题 (FAQ)

Q: uv init 或 uv venv 报错：Failed to download python / invalid peer certificate

问题描述：
某些服务器（特别是内网环境）在尝试下载特定版本的 Python 解释器时，可能会因为网络限制或 SSL 证书过期导致下载失败。
错误日志示例：error: Failed to download ... Caused by: invalid peer certificate: certificate expired...

解决方案：
使用镜像源手动安装指定的 Python 版本（以 3.10.13 为例）：

# 1. 查看可用的 Python 版本
uv python list

# 2. 使用镜像源安装指定版本 (解决证书/网络问题)
uv python install 3.10.13 --mirror https://github-proxy.lixxing.top/https://github.com/astral-sh/python-build-standalone/releases/download

# 3. 安装成功后，再次运行创建环境命令即可
uv venv --python 3.10.13

1.4 项目初始化步骤

配置好 uv 后，请按照以下步骤复现环境：

# 1. 初始化项目 (指定 Python 版本)
# 注意：如果前一步已经手动 install 了 python，这里会直接使用缓存
uv init -p 3.12.4 gdkvm
uv venv --python 3.12.4

# 2. 进入项目目录
cd data/username/Repo/gdkvm

# 3. 激活虚拟环境
source .venv/bin/activate
which python

# 4. 克隆代码仓库
git clone https://github.com/wangrui2025/GDKVM.git

# 5. 同步依赖
uv sync
uv pip check

2. 数据准备

我们使用 CAMUS 和 EchoNet-Dynamic 数据集。请从官方网站下载并按如下方式组织数据： 

./dataset
├── CAMUS
│   ├── training
│   └── testing
└── EchoNet-Dynamic
    ├── FileList.csv
    ├── VolumeTracings.csv
    └── Videos

3. 模型训练

要在 CAMUS 数据集上训练 GDKVM 模型，请运行以下命令。检查点（Checkpoints）将保存在 ./checkpoints 目录下。 

python train.py --dataset camus --batch_size 8 --epochs 100 --lr 1e-4 --save_dir ./checkpoints/camus_gdkvm

4. 推理与评估

要评估预训练模型，请运行： 

python test.py --dataset camus --resume ./checkpoints/camus_gdkvm/best_model.pth --eval_metrics DSC HD95
